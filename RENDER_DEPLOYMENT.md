# DRP Backend ‚Äî Render Deployment Guide

## Quick Deploy (3 steps)

### Step 1: Connect GitHub to Render
1. Go to [dashboard.render.com](https://dashboard.render.com)
2. **New ‚Üí Web Service**
3. Connect GitHub ‚Üí select `Decentralized-Rights-Protocol/Dr-Blockchain`
4. Render auto-detects `render.yaml` ‚Äî all config is pre-set

### Step 2: Set Required Secrets
In the Render dashboard under **Environment**, add:

| Variable | Value |
|---|---|
| `SECRET_KEY` | Auto-generated by Render (click "Generate") |
| `JWT_SECRET_KEY` | Auto-generated by Render |
| `MASTER_ENCRYPTION_KEY` | Auto-generated by Render |

All other env vars are already defined in `render.yaml`.

### Step 3: Deploy
Click **Deploy** ‚Äî Render will:
- Run `pip install -r requirements.render.txt`
- Start `uvicorn api.router:app --host 0.0.0.0 --port $PORT`
- Health check: `GET /health` ‚Üí `{"status": "healthy"}`

## Upgrading from Free Tier

| Need | Solution |
|---|---|
| Persistent storage | Add Render Disk ($7/mo) + set `DRP_DATA_DIR` |
| OrbitDB/IPFS | Enable `DRP_USE_ORBITDB=true` + Node.js bridge |
| Post-quantum crypto | `pip install liboqs-python` (add to requirements.render.txt) |
| ScyllaDB | Add `cassandra-driver` back to requirements |
| Custom domain | Render dashboard ‚Üí Settings ‚Üí Custom Domain |

## Build Command Explained

```bash
# Uses requirements.render.txt (not requirements.txt)
# Strips: cassandra-driver, grpcio, ipfshttpclient, ML libs
# Keeps: fastapi, uvicorn, pydantic, jwt, cryptography, httpx
pip install -r requirements.render.txt
```
